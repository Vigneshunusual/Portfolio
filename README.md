# ⚡ Installation

Clone the repository:

```bash
git clone https://github.com/Vigneshunusual/Portfolio.git
```

Navigate to project folder:

```bash
cd frontend
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm start
```
